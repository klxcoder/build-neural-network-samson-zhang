# Create conda evironment
```bash
conda create -n nn python=3.10
```

# Activate conda environment
```bash
conda activate nn
```

# Init poetry project
```bash
poetry init --no-interaction
```

# Add poetry dependencies
```bash
poetry add numpy pandas matplotlib
```
